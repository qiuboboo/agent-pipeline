## 4. 归因解释

- **图片/视觉导致或相关**：低清、小图、小字、裁切、视觉证据弱、grounding path 缺失/不确定、多图定位等。轻度视觉风险在改写题完整可验时可放行；目标图不可定位、图文冲突、关键视觉条件不可读则不放行。
- **文本/答案标注导致或相关**：选项缺失、答案字母映射不可靠、gold/annotation/rewrite answer 冲突、题干缺关键条件等。选项/答案映射问题常能通过开放题改写规避；题干关键条件缺失或答案真实冲突不能自动规避。
- **metadata/path 导致**：image path、metadata image path、artifact path 不一致，通常属于原数据记录/工程问题；若图片资产存在且题目语义完整，通常可通过规则或改写放行。
- **改写/结构处理导致**：split variant、多子题拆分、rewrite 不完整、agent 不理解等。非 split 且完整可验的改写可放行；split 未确认或 rewrite 不完整继续 hold。
