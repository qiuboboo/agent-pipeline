# ready pass 样本继续分析结论

本轮在初步特征抽取基础上，进一步生成了：

- `ready_deep_dataset_analysis.md/json`：每个数据集的多样性、集中度、类别内统计和代表样本。
- `ready_dataset_diversity.csv`：数据集级多样性指标。
- `ready_category_feature_summary.csv`：数据集-类别级题长、复杂度、multi-step、图像需求等。
- `ready_outlier_samples.csv`：最长题、最高复杂度、最多节点、最多视觉实体样本。
- `figures/*.png`：9 张论文式图表。

## 1. 多样性指标定义

| 指标 | 含义 |
| --- | --- |
| Top1 share | 最大类别占该数据集样本比例，越高说明越单一 |
| Top3 share | 前三大类别占比，衡量是否被少数类型覆盖 |
| entropy_bits | 基于类别分布的信息熵 |
| effective_category_count | $2^{entropy}$，可理解为“等效类别数” |
| HHI | 类别集中度，越高越集中 |
| concentration_label | 根据 Top1 share 粗分为 diverse / moderately_concentrated / strongly_concentrated / single_dominant |

## 5. 总体相关性

| X | Y | Pearson r |
| --- | --- | ---: |
| question_unit_len | complexity_index | 0.8247 |
| multi_step_score | complexity_index | 0.7606 |
| visual_entity_count | multimodal_strength_score | 0.7562 |
| condition_count | node_count | 0.5776 |
| latex_count | answer_unit_len | 0.4349 |

结论：当前复杂度指标主要被题长和 multi-step 分驱动；多模态强度与视觉实体数高度相关。这个结果符合直觉，也说明可以在论文中把 complexity_index 解释为“文本规模 + 推理/视觉结构”的综合 proxy。

## 6. 各数据集分析重点

- `biology/ai2d_biology`：短图示问答，细胞/分子遗传和生态食物网为主；适合展示视觉识别型生物题。
- `biology/scemqa_biology`：样本少但题干长，细胞遗传、生态、解剖生理都有；需要小样本复核。
- `biology/sciverse_biology`：细胞分子遗传占 57.53%，题干较长，很多为实验/图表语境。
- `chemistry/emma_chemistry`：几乎单一 transition-state/SMILES；应标注为专门结构识别子集。
- `chemistry/scemqa_chemistry`：较均衡，化学计量、结构键合、平衡动力学是三大类。
- `chemistry/sciverse_chemistry`：化学中最多样，计量、结构、平衡、热化学、酸碱、有机立体都有覆盖。
- `circuit/eee_bench`：电子电路中最多样，网络分析、模拟电子、数字逻辑、信号系统、控制系统都有。
- `geography/ai2d_geography`：短图示题，月相/地球太阳月亮、图表识别为主。
- `geography/geosqa`：整体最均衡；气候、人口经济、图表、地图地形、水文、资源环境都有。
- `math/cmm_math`：文本代数为主，线性规划/约束最值占 38.03%，几乎不依赖图。
- `math/geometry3k`：短几何图题，角追主导，其他包括圆弧弦、面积周长、变量求解。
- `math/geoqa_plus`：视觉几何为主，但比 `mm_math` 稍多样，含空间拼图、数论、组合计数。
- `math/mathvision`：几何为主，但题型比 `mm_math` 更宽，含计数、空间、数论、时间单位。
- `math/mm_math`：高度集中在几何角追，适合专门评估几何视觉推理。
- `math/scemqa_math`：样本量小但类别均衡，统计、代数、几何都有。
- `physics/emma_physics`：力学主导，辅以电磁；题干较长。
- `physics/multi_physics`：中文物理图题，力学和电磁占大头。
- `physics/physreason`：长链路物理推理，力学极度集中，复杂度最高。
- `physics/phyx`：物理中较均衡，力学、光学、热/流体、电磁都有。
- `physics/sciverse_physics`：力学主导，样本量较小。
- `physics/seephys`：力学、电磁、光学为主，较适合作为综合物理视觉推理集。

## 7. 建议的下一步精分类

1. 每个数据集保留 8-15 个最终类别；超过 15 类不利于论文叙述。
2. 对 Top1 占比超过 80% 的数据集，不必强拆太细，应在报告中强调“专门性”。
3. 对 `other` 仍超过 10% 的数据集做人工/LLM 复核：`ai2d_biology`、`eee_bench`、`geometry3k`、`geoqa_plus`、`scemqa_math`、`multi_physics`、`phyx`、`sciverse_physics`、`seephys`。
