# Ready PPT HTML 静态幻灯片

入口文件：

`index.html`

## 使用方法

1. 直接用浏览器打开 `index.html`。
2. 使用键盘 `←` / `→` 切换页面。
3. 按 `P` 或浏览器打印，可导出 PDF。

## 图片素材

统计图放在：

`assets/figures/`

当前使用：

- `subject_pie.png`
- `category_bar.png`

备用图：

- `question_length_boxplot.png`
- `complexity_scatter.png`
- `multistep_histogram.png`
- `visual_type_bar.png`

如果后续让 Gemini 只生成更漂亮的统计图，把新图替换到 `assets/figures/` 下的同名文件即可。
