## 5. 当前这轮修复后批跑能确认的结论

1. `multi_physics` 的 `picture` 字段图片已经能被正确接入，旧版漏图问题已被消除。
2. Geometry3K 的多图接入、ROI 和图文对齐链路能稳定覆盖一批样本，但仍存在相当比例的对齐失败样本。
3. SCEMQA 主要是开放题保留为 `keep_open`；CMM-Math 与部分 EEE-Bench 主要体现 `blank_open / split_open`；纯图编号题会走 `drop_image_index`。
4. `pass / review / reject` 三类门控都有真实样本支撑，并且都能在逐样本报告中追溯到原文本、原图、改写结果和原因码。

## 6. 剩余风险

- PhysReason 仍然不可用，本轮没有拿到可跑的结构化样本。
- Geometry3K 仍有较高 reject 比例，说明图文对齐与稳定推理路径仍是主要瓶颈。
- `split_open` 依然是最主要的人审入口，拆题后目标是否合理仍需人工复核。
