# eeebench00001 的 method_2 中文版步骤说明（2026-04-16）

本文只描述 `eeebench00001` 里 `method_2` 这条解法本身，不讨论 claim extraction、gate fail 或 debug 细节。

## method_2 的核心思路

这条方法不是先做 K-map 分组，而是先直接观察每个平面里 `1` 的分布模式，再把不影响结果的变量消去。

一句话概括：

- 先看激活 pattern
- 再消去变化变量
- 最后写出两个乘积项并相加

## 中文版步骤

### 第 1 步：把图分成两个平面来看

这张 5 变量 K-map 分成两个 `4x4` 平面：

- 左边平面对应 `X=0`
- 右边平面对应 `X=1`

因此可以分别分析两个平面中 `1` 的分布规律。

### 第 2 步：观察左边 `X=0` 平面的 `1` 分布

在左边平面中，所有 `1` 都只出现在：

- 列 `PQ=00` 和 `PQ=10`
- 行 `RS=01` 和 `RS=11`

也就是说，左边平面的 `1` 只出现在某一组固定条件下。

### 第 3 步：从左边平面中找出保持不变的变量

先看列：

- `PQ=00` 和 `PQ=10` 这两列里，`Q` 一直等于 `0`
- `P` 会变化，所以 `P` 不是决定条件

再看行：

- `RS=01` 和 `RS=11` 这两行里，`S` 一直等于 `1`
- `R` 会变化，所以 `R` 不是决定条件

又因为整个左边平面就是 `X=0`，所以 `X=0` 也必须保留。

因此左边平面对应的条件是：

- `Q' S X'`

### 第 4 步：观察右边 `X=1` 平面的 `1` 分布

在右边平面中，所有 `1` 都只出现在：

- 列 `PQ=01` 和 `PQ=11`
- 行 `RS=00` 和 `RS=10`

说明右边平面的激活也满足另一组固定条件。

### 第 5 步：从右边平面中找出保持不变的变量

先看列：

- `PQ=01` 和 `PQ=11` 这两列里，`Q` 一直等于 `1`
- `P` 会变化，所以 `P` 被消去

再看行：

- `RS=00` 和 `RS=10` 这两行里，`S` 一直等于 `0`
- `R` 会变化，所以 `R` 被消去

又因为整个右边平面就是 `X=1`，所以 `X=1` 必须保留。

因此右边平面对应的条件是：

- `Q S' X`

### 第 6 步：把两个平面的结果相加

左边平面得到：

- `Q' S X'`

右边平面得到：

- `Q S' X`

因此最简 SOP 为：

- `Q' S X' + Q S' X`

## 为什么这叫 variable-elimination 方法

因为这条方法的重点不是“圈出一个 group”，而是看哪些变量在所有激活位置中其实不影响结果。

在这题里：

- `P` 在左边和右边对应位置中都会变化
- `R` 也会变化
- 但这些变化不会改变哪里是 `1`

所以：

- `P` 可以消去
- `R` 可以消去

最后只保留真正决定激活的变量：

- 左边保留 `Q' S X'`
- 右边保留 `Q S' X`

## 一句话总结

`method_2` 的做法是：分别观察 `X=0` 和 `X=1` 两个平面的 `1` 分布，找出在每个平面中始终不变的变量，消去会变化的 `P` 和 `R`，最后得到：

- `Q' S X' + Q S' X`
